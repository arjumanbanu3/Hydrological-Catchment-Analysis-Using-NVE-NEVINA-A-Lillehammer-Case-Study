# Hydrological Catchment Analysis Using NVE NEVINA: A Lillehammer Case Study

## Overview

This project demonstrates a hydrological analysis workflow using NVE's NEVINA platform. A catchment near Lillehammer, Norway, was delineated and characterized using national hydrological datasets.

The project includes:

- Catchment delineation
- Catchment characterization
- Low-flow analysis
- Flood-frequency estimation
- Interpretation of hydrological data

The objective is to demonstrate practical hydrological and GIS skills relevant to watershed assessment and flood-risk analysis.

---

# Study Area

The study area is located near Lillehammer in Innlandet County, Norway, within the Vorma–Lågen river system.

A catchment upstream of a selected outlet point was generated using the NVE NEVINA platform and used as the basis for all analyses.

# Discussion

The catchment covers a large portion of the Vorma–Lågen river system and spans substantial elevation gradients, indicating diverse hydrological conditions and significant snowmelt influence.

Low-flow analysis suggests moderate groundwater contribution to streamflow, while flood-frequency analysis highlights progressively increasing flood magnitudes with increasing return period.

The estimated Q200 flood exceeds 2200 m³/s, demonstrating the importance of flood-risk assessment in large Norwegian catchments.
